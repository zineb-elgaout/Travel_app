# Travel App 🌍

A modern, full-featured travel application built with React Native and Expo, designed to help users discover, plan, and share their travel experiences.

## Features

- 📍 **Destination Discovery** - Browse and explore popular travel destinations
- 🗺️ **Trip Planning** - Create and organize your travel itineraries
- 💾 **Save Favorites** - Bookmark destinations and trips for later
- 📸 **Photo Gallery** - Share and view travel photos
- 🌐 **Universal Platform** - Works seamlessly on iOS, Android, and web

## Tech Stack

- **Framework**: [Expo](https://expo.dev) & React Native
- **Language**: TypeScript
- **Build Tool**: [create-expo-app](https://www.npmjs.com/package/create-expo-app)
- **Routing**: [Expo Router](https://docs.expo.dev/router/introduction) (file-based)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Expo CLI (optional, but recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/zineb-elgaout/Travel_app.git
   cd Travel_app/my-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npx expo start
   ```

4. **Open the app**
   - Press `i` for iOS Simulator
   - Press `a` for Android Emulator
   - Press `w` for Web
   - Scan the QR code with [Expo Go](https://expo.dev/go) app on your phone

## Project Structure

```

├── app/                  # Main app directory with file-based routing
├── components/           # Reusable React components
├── assets/              # Images, icons, and other media
├── utils/               # Utility functions and helpers
```

## Available Scripts

- `npm start` - Start the Expo development server
- `npm run reset-project` - Reset to a fresh project state
- `npm run ios` - Build and run on iOS simulator
- `npm run android` - Build and run on Android emulator
- `npm run web` - Run on web browser
